Atividade-Gerenciamente-de-Estoque

Sistema de gerenciamento de Estoque.

Descrição pormenorizada do sistema Sistema de gerenciamento escolar, que visa os seguintes objetivos: * O sistema deve possibilitar o cadastro de clientes, fornecedores e produtos. * O sistema deve exibir relatórios de clientes, fornecedores e produtos. * O sistema deve exibir um relatório com todos os produtos de um fornecedor. * O sistema deve exibir uma compra feita por um cliente, calculando o valor total da compra mediante a quantidade de produtos comprados.
